# תיקיית Metadata

תיקייה זו מכילה נתוני מטא-דאטה של הפרויקט, כולל רשימת הישובים בישראל.

## קבצים

### `settlements_list.geojson` ⭐ מקור יחיד
קובץ GeoJSON המכיל את כל הישובים בישראל עם קואורדינטות גיאוגרפיות.

**מבנה**:
- כל feature מכיל:
  - `MGLSDE_LOC` - שם הישוב בעברית
  - `geometry` - קואורדינטות (Point או Polygon)
  - שדות נוספים (קוד ישוב, סוג ישוב, וכו')

הקוד ב-`src/validation.py` טוען את הרשימה ישירות מקובץ זה.

### `settlements_coordinates.json`
קובץ JSON המכיל את כל הישובים עם קואורדינטות (נוצר אוטומטית מה-GeoJSON).

### `DATA_SOURCES.md`
מסמך מפורט על מקורות הנתונים הטובים ביותר ליצירת רשימה מקיפה של ישובים בישראל כולל קואורדינטות.

## עדכון הרשימה

כדי לעדכן את רשימת הישובים:

1. עדכן את `settlements_list.geojson` עם ישובים חדשים
2. הקוד יטען את הרשימה אוטומטית בעת הפעלה

## מקורות מומלצים

- **settlements_list.geojson** - מקור יחיד נוכחי
- **odata.org.il** - מאגר כתובות בישראל עם קואורדינטות
- **Nominatim API** - geocoding חינמי
- **הלמ"ס** - נתונים רשמיים

ראה `DATA_SOURCES.md` לפרטים מלאים.
